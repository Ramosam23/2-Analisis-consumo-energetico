# Src – Código Reutilizable

Esta carpeta está reservada para funciones y scripts reutilizables relacionados con el análisis y modelado del consumo energético.

## Estado actual

En este proyecto, todo el análisis, preparación de datos y modelado se realizó **de forma explícita dentro del notebook**, priorizando la claridad, trazabilidad y valor educativo del proceso.

Por este motivo, no se implementaron módulos adicionales en esta carpeta.

## Uso futuro

En una evolución del proyecto, esta carpeta podría incluir:
- Funciones para preparación de datos.
- Utilidades para creación de variables rezagadas.
- Funciones de evaluación de modelos.
- Pipelines reutilizables para nuevos datasets de series temporales.
