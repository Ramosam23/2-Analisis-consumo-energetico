# Raw Data – Consumo Energético

Esta carpeta contiene el dataset original de consumo energético utilizado en el proyecto.

## Descripción del dataset

El conjunto de datos corresponde a mediciones de consumo eléctrico registradas a lo largo del tiempo, con variables relacionadas con:

- Consumo activo y reactivo
- Voltaje
- Intensidad de corriente
- Sub-mediciones por zonas o circuitos

Los datos incluyen marcas temporales (fecha y hora) que permiten el análisis de patrones temporales, estacionales y la construcción de modelos de series de tiempo.

## Uso de los datos

Los archivos contenidos en esta carpeta:

- **No han sido modificados**
- Se utilizan únicamente como fuente de entrada
- No deben alterarse para preservar la integridad del análisis

Todas las transformaciones, limpiezas y agregaciones se realizan y almacenan en la carpeta `processed/`.
